# vavi-sound-dx7

DX7 emulated synthesizer. `javax.sound.midi.spi` compatible.

## install

 * maven repo: [jitpack]()
 * copy [`unpacked.bin`](https://github.com/bwhitman/learnfm/blob/f5415157c65b0298dad692e5e332c71644718e28/unpacked.bin) into your class path
